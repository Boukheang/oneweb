# OneWeb - One Piece Fan Landing Page

A responsive One Piece fan landing page built from a Bootstrap-based template.
This project showcases the Straw Hat Pirates, popular arcs, and a contact section with a map.

## What was improved

- Cleaned up HTML structure and fixed broken markup.
- Added responsive styling for mobile, tablet, and desktop screens.
- Improved hero section and card spacing for a more polished layout.
- Fixed the portfolio lightbox links and map iframe attributes.
- Added accessibility enhancements including proper navigation toggle labeling.

## Files

- `index.html` - Main landing page content and structure.
- `assets/css/style.css` - Primary stylesheet with responsive layout improvements.
- `assets/js/main.js` - Page interaction scripts for mobile navigation, portfolio filter, and scroll behavior.

## Preview locally

1. Open `index.html` directly in your browser.
2. For the best results, serve the folder from a local web server.
   Example using Python 3:
   ```bash
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` in your browser.

## Notes

- The page uses Bootstrap 5 and vendor assets included in the `assets/vendor` folder.
- The contact section uses Google Maps embed and a simple contact form placeholder.
- If you want to customize content, replace the text and image paths in `index.html`.
